Variables

const

Una variable declarada como const no puede ser reasignada, amenos de que se guarde un objecto

let

let solo se puede declarar en un scope y solo es accesible desde ese scope

var

Es una variable global que puede ser accesible de distintos scopes

Array methods

  • Que es un array? - Es una varibale que agrupa elementos

    Es una propiedad que inserta al incio del arreglo el elemento que le pasamos a la funcion

  • Las funciones que mutan el arreglo, se les conoce como imperativas
  • Las funciones que no muatn el arreglo y devuelven un nuevo valor, se les concoe como declarativas

  • Para eliminar el ultimo de un arreglo de usa el metodo pop

  • Para eliminar el Inicio de un arreglo de usa el metodo shift

  • Foreach Es un metodo que ayuda a iterar los elementos de un arreglo de forma mas rapida

  • map Es un metodo que ayuda a iterar los elementos de un arreglo pero devuelve un nuevo arreglo

    Solo se encarga de ver que el elemento exista en un arreglo tradicional

  • .some

    Se encarga de ver que el elemento exista en un arreglo de objectos

  • .findIndex

    Es un array method que ayuda en contrar un elemento por el index

    Es un array method que se encraga de sumar todos los elementos, es muy util con los carritos de la compra

  • .filter

    Es un array method que recibe como argumento un callabck y dentro se la pasa una condicion. Este array method retorna un nuevo arrey

  • .find

    Es un array method que recibe por parametro una condicion de busqua h retorna todo el elemento

  • .every

    Es un array method que se encarga de comprobar que todos los elementos del arreglo cumplan con una condicion que se especifican

  • .some

    Comprueba que almenos uno de los elementos cumpla con la condicion
